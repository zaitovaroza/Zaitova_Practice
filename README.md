# Учет партнеров компании "Мастер пол"

## Разработчик
Заитова Роза, группа ИПО-41

## Описание проекта
Разработка подсистемы для работы с партнерами в рамках производственной практики.

## Функционал
- Просмотр списка партнеров
- Добавление/редактирование данных о партнере
- Просмотр истории продаж
- Автоматический расчет скидки

## Технологии
- C# / .NET 8.0
- WPF
- Entity Framework Core
- PostgreSQL
- MSTest
